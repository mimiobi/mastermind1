Deploy a High-Availability Web App using CloudFormation

In this project, you will deploy a High-availability Web App using 
CloudFormation to AWS. 

The following are files you will need to achieve this:

infraDiagramU - Architectural diagram for this project
ptwoinfra.yaml - scripts for the network infrasructure for the web app
ptwoinfra-params.json - parameters for the network infrastructure
ptwoservers.yaml - scripts for the server for the web app
ptwoservers-params.json- parameters for the server 
create.sh - AWS CloudFormation create-stack
update.sh - AWS CloudFormation update-stack

https://ptwoudagram.s3-us-west-2.amazonaws.com/index.html - URL Link to 
show it works!
